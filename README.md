Open-Source, privacy-first alternative to standard web search engines. Instead of querying a remote server every time you look for a website, this project allows you to query a curated, highly structured meta-database of valuable internet domains, communities, and personal web spaces directly on your local device.

It is designed for tech enthusiasts, privacy advocates, and anyone annoyed by search engine malvertising, SEO spam, and content farms.

# Features

 - Instant search across a large collection of well-known domains
 - Simple and lightweight interface
 - Fast local search with no waiting for server responses
 - Privacy-friendly: no network requests required for searching

# Why Use Offline Web Search?

Finding a website can be faster than opening a browser and typing a full URL. Offline Web Search provides a searchable index of popular domains directly on your device.

# Permissions

The app requires minimal permissions and does not rely on remote services for searching.

# Databases

The application is highly customizable and allows you to load different databases directly from a file. You can curate your own search indexes, share them, or swap between specialized databases depending on your current needs—all completely offline.

# Open Source

This project is open source and welcomes contributions, bug reports, and suggestions.
